PowerShell Commands

OS Information

Using the following PowerShell command will give you detailed operating system information.

Get-ComputerInfo
CPU Information

Using the following PowerShell command will give you information about the system processor.

Get-CimInstance Win32_Processor
RAM Information

Using the following PowerShell command will give you details about installed physical memory.

Get-CimInstance Win32_PhysicalMemory
Storage Information

Using the following PowerShell command will give you information about physical storage disks.

Get-PhysicalDisk
GPU Information

Using the following PowerShell command will give you details about the video controller.

Get-CimInstance Win32_VideoController
